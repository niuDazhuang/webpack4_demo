# wtwd-app-h5

- webpack4多页应用

> 依赖
- webpack@4
- highcharts
- art-template
- less

## install

```
$ cnpm i
```

## dev

```bash
$ npm start

# => http://localhost:8080/record.html
# => http://localhost:8080/statistic.html
# => http://localhost:8080/xinlv.html
```

## build

```
$ npm run build
```
